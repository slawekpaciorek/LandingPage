Początkujący programista z wielką pasją!

Cześć! Programowaniem interesuję się od niecałych 2 lat, mam już za sobą pierwsze projekty edukacyjne wykorzystujące Javę oraz technologie pokrewne, a także pierwszą stronę komercyjną, zaprojektowaną i wykonaną dla pracowni projektowania wnętrz. Ostatnimi czasy do pracy programisty przygotowywałem się na bootcampie programistycznym trwającym ponad pół roku. W trakcie nauki przerobiłem zagadnienia dotyczące Javy SE, Javy EE, Hiberante, SQL, HTML, CSS, JavaScript i JQuery, Linux, Git, Maven, HTTP. W trakcie zajęć miałem okazję pracować nad projektem grupowym w tehnologii SCRUM. Podczas naszej pracy miała powstać aplikacja webowa dzięki kórej klient będzie miał możliwość zamówienia części do wybranego przez siebie pojazdu. Aplikacja korzysta z zewnętrzengo API w którym zawarte są infomracje na temat pojazdów i dostępnych części. Budowanie aplikacji było podzielone na 3 sprinty w których kolejno powstawała aplikacjia konsolowa, aplikacja webowa korzystająca z aplikacji konsolowej jako API, oraz aplikacja restowa zbierająca statystyki. Cały projekt został oparty o następujące technologie : Java 8, Java EE, Hibernate, Maven, REST, JQuery, Ajax, HTML, CSS, Bootsrtap. Chcesz zobaczyć moje prace? Zerknij do portfolio.
