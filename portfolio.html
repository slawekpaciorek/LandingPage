Employee Contract Manager

Prosty projekt webowy do zarządzania kontraktami pracowników, użytkownik może wykonywać prostę operacje CRUD. Projekt powstał w opraciu o następujące technologie : Java 8, JavaEE, Jboss, Maven, HTML, CSS. Projekt nie posiada podłączenia do bazy danych. Kod żródłowy można obejrzeć klikając w link.

User Engine App

Prosty projekt webowy do zarządzania użytkownikami systemu, klient może wykonywać proste operacje CRUD. Aplikacja wykonana przy użcyiu : Java 8, JavaEE, MySQL, Hibernate, JBoss, CSS, JQuery, HTML. Projekt posiada podłączenie do bazy danych. Kod żródłowy można obejrzeć klikając w link.

File Analyzer

Prosta aplikacja konsolowa analizująca zapisany plik CSV. Aplikacja stworzona na potrzeby edukacyjne. Podczas pisania aplikacji dążyłem do opanowania podstaw posługiwania się "stremami" w Java 8. Kod żródłowy można obejrzeć klikając w link.

Wnetrznosci - Aneta Stokowska

Landing Page wykonany dla pracownii architektury wnętrz Anety Stokowskiej. Głównym celem strony jest przedstawienie wykonanych wizualizacji oraz kompletnie przeprowadzonych projektów. Użytkownik może również zasięgnąć podstawowych informacji o projektantce oraz ofercie współpracy. W zakładce kontakty został utworzony formularz ułatwiający kontakt potencjalnemu klientowi. Projekt został oparty o popularny system CMS, Wordpress. W trakcie tworzenia strony, wykorzystano następujące technologie : PHP, HTML, CSS, JQuery, Bootstrap. Aktualną wersję projektu można zobaczyć pod adresem www.wnetrznosci.com.pl
